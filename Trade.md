# **무한매수법 V4.0 실행 정리**

이 문서는 네이버 카페 글 `무한매수법 V4.0 방법론 _ 1.일반모드`와 `2.소진후 리버스모드` 내용을 바탕으로, 실제 매수/매도 행동 관점에서 보기 쉽게 요약한 실행 메모입니다.

투자 판단과 손실 책임은 투자자 본인에게 있습니다. 아래 내용은 방법론 정리이며 투자 권유가 아닙니다.

## **1. 전체 흐름**

1. 원금과 분할 수를 먼저 정한다.
2. 보유수량이 0이면 일반모드의 첫 매수부터 시작한다.
3. 매일 `T값`, 잔금, 평단, 별지점을 계산한다.
4. 일반모드에서는 정해진 방식으로 LOC 매수, LOC 매도, 지정가 매도를 건다.
5. 잔여 회차가 1회 미만이 되면 소진 상태로 보고 리버스모드로 전환한다.
6. 리버스모드에서는 일부 보유분을 팔아 현금을 만들고, 5거래일 평균 종가 기준으로 매수/매도를 반복한다.
7. 종가가 회복 조건을 만족하면 일반모드로 복귀한다.
8. 보유수량이 0이 되면 한 사이클 종료로 본다.

## **2. 시작 전 설정**

### **종목**

공식적으로 언급된 종목은 다음 두 가지입니다.

| **종목** | **특징** |
| --- | --- |
| TQQQ | 초심자 우선 추천 |
| SOXL | 변동성이 더 크고 기대수익도 크지만 난도가 높음 |

처음 시작한다면 TQQQ로 변동성에 익숙해진 뒤 SOXL을 고려하는 순서입니다.

### **원금**

무한매수법에 사용할 원금을 먼저 정합니다.

정한 원금은 매수 후 잔금이 남아도 다른 투자에 사용하지 않습니다.

TQQQ와 SOXL을 동시에 운용할 경우 각 종목별 원금을 독립적으로 정합니다.

### **분할 수**

권장 범위는 보통 `20분할`, `30분할`, `40분할`입니다.

| **분할 수** | **성향** |
| --- | --- |
| 20분할 | 공격적, 위험도 높음 |
| 30분할 | 중간 |
| 40분할 | 방어적, 안정성 높음 |

1회 매수금의 시작값은 다음과 같습니다.

```
초기 1회 매수금 = 원금 / 분할 수
```

## **3. 핵심 용어**

### **T값**

`T`는 진행 회차입니다.

```
1.0T = 1회분이 진행된 상태
```

일반모드에서의 T 계산은 다음과 같습니다.

| **상황** | **T 계산** |
| --- | --- |
| 1회 매수 체결 | T + 1 |
| 절반 매수 체결 | T + 0.5 |
| 쿼터매도 체결 | 직전 T × 0.75 |
| 지정가매도 후 LOC 매수 | 직전 T × 0.25 + 체결 매수분 |

지정가매도 후 LOC 매수가 체결되면 다음처럼 계산합니다.

```
1회 매수 체결 시: 직전 T × 0.25 + 1
절반 매수 체결 시: 직전 T × 0.25 + 0.5
```

T값은 소수점 제한 없이 그대로 관리합니다.

### **별지점**

일반모드에서 별지점은 매수와 매도를 가르는 기준 가격입니다.

```
별지점 = 평단 × (1 + 별%)
```

별% 공식은 다음과 같습니다.

| **종목/분할** | **별% 공식** |
| --- | --- |
| 20분할 TQQQ | (15 - 1.5T)% |
| 40분할 TQQQ | (15 - 0.75T)% |
| 20분할 SOXL | (20 - 2T)% |
| 40분할 SOXL | (20 - T)% |

매수와 매도가 같은 가격에 겹치지 않도록 매수점은 별지점보다 `0.01` 낮게 둡니다.

```
매수점 = 별지점 - 0.01
매도점 = 별지점
```

## **4. 매일 계산할 값**

매일 주문 전 다음 값을 갱신합니다.

1. 현재 보유수량
2. 현재 평단
3. 현재 잔금
4. 현재 T값
5. 오늘의 1회 매수금
6. 오늘의 별%
7. 오늘의 별지점
8. 현재 모드가 일반모드인지 리버스모드인지

일반모드의 1회 매수금은 매일 다시 계산합니다.

```
20분할 1회 매수금 = 잔금 / (20 - T)
40분할 1회 매수금 = 잔금 / (40 - T)
```

## **5. 일반모드 실행법**

일반모드는 소진 전까지의 기본 운용 구간입니다.

### **일반모드 구간**

| **분할 수** | **전반전** | **후반전** | **리버스모드 전환** |
| --- | --- | --- | --- |
| 20분할 | T < 10 | 10 <= T <= 19 | T > 19 |
| 40분할 | T < 20 | 20 <= T <= 39 | T > 39 |

### **첫 매수**

보유수량이 0이고 `T = 0`이면 첫 매수를 시작합니다.

행동은 다음과 같습니다.

1. 초기 1회 매수금을 계산한다.
2. 현재가보다 충분히 높은 가격에 LOC 매수를 건다.
3. 하락 체결을 대비해 아래 가격대에도 LOC 매수를 분산해서 건다.
4. 체결 결과에 따라 T값과 잔금을 갱신한다.

첫 매수는 보유를 시작하기 위한 단계라, 기본적으로 1회 매수를 시도합니다.

### **전반전 매수**

전반전은 T가 전체 분할 수의 절반 미만인 구간입니다.

행동은 다음과 같습니다.

1. 오늘의 1회 매수금을 계산한다.
2. 1회 매수금의 절반은 `별지점 - 0.01`에 LOC 매수로 건다.
3. 나머지 절반은 평단에 LOC 매수로 건다.
4. 큰 하락을 대비해 더 낮은 가격에도 LOC 매수를 추가로 분산한다.
5. 모든 매수 주문은 LOC로 처리한다.

전반전 매수 구조는 다음과 같습니다.

```
절반: 별지점 - 0.01 LOC 매수
절반: 평단 LOC 매수
추가: 더 낮은 가격대 LOC 매수
```

### **후반전 매수**

후반전은 T가 전체 분할 수의 절반 이상인 구간입니다.

이 구간부터 별%가 마이너스가 될 수 있어 별지점이 평단보다 아래로 내려갑니다.

행동은 다음과 같습니다.

1. 오늘의 1회 매수금을 계산한다.
2. 1회 매수금 전체를 `별지점 - 0.01`에 LOC 매수로 건다.
3. 큰 하락을 대비해 더 낮은 가격에도 LOC 매수를 추가로 분산한다.
4. 모든 매수 주문은 LOC로 처리한다.

후반전 매수 구조는 다음과 같습니다.

```
전체: 별지점 - 0.01 LOC 매수
추가: 더 낮은 가격대 LOC 매수
```

### **일반모드 매도**

일반모드 매도는 전반전과 후반전 모두 동일합니다.

행동은 다음과 같습니다.

1. 보유수량의 1/4을 계산한다.
2. 1/4 수량은 별지점에 LOC 매도로 건다.
3. 나머지 수량은 목표수익률 가격에 지정가 매도로 건다.
4. 지정가 매도는 프리장부터 애프터장까지 최대한 길게 활용한다.
5. 주간장은 사용하지 않는다.

목표수익률은 다음과 같습니다.

| **종목** | **지정가 매도 목표** |
| --- | --- |
| TQQQ | 평단 대비 +15% |
| SOXL | 평단 대비 +20% |

일반모드 매도 구조는 다음과 같습니다.

```
1/4 수량: 별지점 LOC 매도
나머지 수량: TQQQ +15% 또는 SOXL +20% 지정가 매도
```

### **일반모드 종료**

보유수량이 0이 되면 한 사이클이 종료된 것으로 봅니다.

사이클 종료 후에는 다시 첫 매수로 돌아갑니다.

수익금을 원금에 포함할지는 복리/단리 선택입니다.

## **6. 리버스모드 실행법**

리버스모드는 일반모드에서 잔여 회차가 1회 미만이 된 뒤 적용하는 구간입니다.

### **리버스모드 진입 조건**

| **분할 수** | **진입 조건** |
| --- | --- |
| 20분할 | T > 19 |
| 40분할 | T > 39 |

남은 현금이 일부 있어도 위 조건이면 리버스모드로 전환합니다.

외부 자금은 추가 투입하지 않습니다.

### **리버스모드 첫날**

첫날은 매수하지 않고 매도만 합니다.

| **분할 수** | **첫날 매도 수량** | **주문 방식** |
| --- | --- | --- |
| 20분할 | 보유수량의 1/10 | MOC 매도 |
| 40분할 | 보유수량의 1/20 | MOC 매도 |

수량은 소수점이 나오면 내림 처리합니다.

### **리버스모드 별지점**

리버스모드의 별지점은 일반모드 공식과 다릅니다.

```
리버스모드 별지점 = 직전 5거래일 종가 평균
```

둘째 날부터 이 별지점을 기준으로 매수와 매도를 판단합니다.

### **리버스모드 둘째 날 이후**

기준 행동은 다음과 같습니다.

1. 직전 5거래일 종가 평균으로 별지점을 계산한다.
2. 현재 보유수량과 잔금을 확인한다.
3. 별지점 위에서는 매도 관점으로 대응한다.
4. 별지점 아래에서는 매수 관점으로 대응한다.
5. 매도로 생긴 현금은 이후 매수 가능 잔금에 포함한다.

### **리버스모드 매도**

매도 수량은 매일 직전 보유수량 기준으로 계산합니다.

| **분할 수** | **매도 수량** |
| --- | --- |
| 20분할 | 직전 보유수량의 1/10 |
| 40분할 | 직전 보유수량의 1/20 |

수량은 내림 처리합니다.

T값은 다음처럼 조정합니다.

```
20분할 매도 후 T = 직전 T × 0.9
40분할 매도 후 T = 직전 T × 0.95
```

### **리버스모드 매수**

둘째 날부터 매수는 현재 잔금의 1/4만큼 시도합니다.

```
리버스모드 매수 시도금 = 현재 잔금 × 0.25
```

잔금에는 기존 현금과 리버스모드 매도로 확보한 현금이 모두 포함됩니다.

T값은 다음처럼 조정합니다.

```
20분할 매수 후 T = 직전 T + (20 - 직전 T) × 0.25
40분할 매수 후 T = 직전 T + (40 - 직전 T) × 0.25
```

### **리버스모드 종료 조건**

종가가 평단 대비 일정 수준 이상 회복되면 리버스모드를 종료하고 일반모드로 복귀합니다.

| **종목** | **일반모드 복귀 기준** |
| --- | --- |
| TQQQ | 종가가 평단 대비 -15%보다 위 |
| SOXL | 종가가 평단 대비 -20%보다 위 |

일반모드로 복귀할 때는 리버스모드에서 이어진 T값을 그대로 사용합니다.

복귀 후 1회 매수금은 다시 일반모드 공식으로 계산합니다.

```
20분할 1회 매수금 = 잔금 / (20 - T)
40분할 1회 매수금 = 잔금 / (40 - T)
```

## **7. 매일 실행 체크리스트**

### **장 시작 전 또는 주문 전**

1. 현재 모드를 확인한다.
2. 보유수량을 확인한다.
3. 평단을 확인한다.
4. 잔금을 확인한다.
5. T값을 확인한다.
6. 일반모드라면 일반모드 별지점을 계산한다.
7. 리버스모드라면 직전 5거래일 평균 종가를 계산한다.
8. 오늘의 매수 가능 금액을 계산한다.
9. 오늘의 매도 수량을 계산한다.

### **일반모드 주문**

1. 전반전인지 후반전인지 확인한다.
2. 전반전이면 별지점 LOC 매수와 평단 LOC 매수를 나누어 건다.
3. 후반전이면 별지점 LOC 매수 중심으로 건다.
4. 추가 하락 대비용 LOC 매수를 아래 가격대에 분산한다.
5. 보유수량의 1/4을 별지점 LOC 매도로 건다.
6. 나머지 수량은 TQQQ +15%, SOXL +20% 지정가 매도로 건다.

### **리버스모드 주문**

1. 첫날이면 매수 없이 정해진 비율만 MOC 매도한다.
2. 둘째 날부터는 직전 5거래일 평균 종가를 기준으로 삼는다.
3. 매도 조건이면 정해진 비율만큼 매도한다.
4. 매수 조건이면 현재 잔금의 1/4만큼 매수 시도한다.
5. 체결 후 T값, 잔금, 보유수량, 평단을 갱신한다.
6. 종가가 복귀 조건을 만족하는지 확인한다.

### **체결 후 기록**

1. 체결된 매수 수량과 금액을 기록한다.
2. 체결된 매도 수량과 금액을 기록한다.
3. 잔금을 갱신한다.
4. 보유수량을 갱신한다.
5. 평단을 갱신한다.
6. T값을 갱신한다.
7. 다음 날 모드를 결정한다.

## **8. 모드 판단 요약**

```
보유수량 = 0
=> 사이클 종료, 첫 매수부터 재시작

일반모드 20분할에서 T <= 19
=> 일반모드 유지

일반모드 40분할에서 T <= 39
=> 일반모드 유지

20분할에서 T > 19
=> 리버스모드 전환

40분할에서 T > 39
=> 리버스모드 전환

리버스모드 중 TQQQ 종가 > 평단 × 0.85
=> 일반모드 복귀

리버스모드 중 SOXL 종가 > 평단 × 0.80
=> 일반모드 복귀
```

## **9. 주문 방식 요약**

| **상황** | **주문 방식** |
| --- | --- |
| 일반모드 매수 | LOC 매수 |
| 일반모드 별지점 매도 | LOC 매도 |
| 일반모드 최종 목표 매도 | 지정가 매도 |
| 리버스모드 첫날 매도 | MOC 매도 |
| 리버스모드 둘째 날 이후 | 별지점 기준 매수/매도 |

## **10. 주의사항**

정한 원금 외 추가 투입은 하지 않는 것을 원칙으로 합니다.

매수 후 남은 잔금도 다른 투자에 사용하지 않습니다.

주간장은 사용하지 않는 것으로 정리되어 있습니다.

가격 제한 때문에 주문이 거부될 수 있으므로, 큰 하락장에서는 증권사별 주문 가능 범위를 확인해야 합니다.

TQQQ와 SOXL을 동시에 운용할 경우 원금, T값, 평단, 잔금, 모드를 각각 독립적으로 관리해야 합니다.

같은 종목을 서로 다른 분할 수로 동시에 운용한다면 계좌를 분리하는 편이 관리상 유리합니다.
